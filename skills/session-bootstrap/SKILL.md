# 会话启动引导

## 目标

在不全量读取全部仓库文档的前提下，用最小必要文档启动新会话并恢复最新项目状态。

## 强制读取顺序

1. `/AGENT.md`
2. `/CLAUDE.md`
3. `/docs/indexes/00-总索引.md`
4. `/docs/indexes/02-当前阶段索引.md`
5. `/docs/indexes/03-当前任务索引.md`

## 输出要求

在进入更深层工作前，Agent 应当能够回答：

- 当前项目处于哪个阶段
- 当前主任务是什么
- 本次会话还需要额外读取哪些文档
- 当前任务是否已有边界和进入门禁
- 当前阶段是否已有自测边界

## 停止条件

如果在最小读取集之后，当前任务仍然不清晰，则只允许继续读取 `docs/context/` 或 `docs/plans/` 中最小相关文件，不得直接全量扩读。
