## 4. 过程中修正的问题

- PowerShell 脚本中的中文提示在 `powershell.exe` 下存在 UTF-8 兼容风险，已改为 ASCII 提示，避免一键脚本解析异常。
- `.cmd` 包装器中的中文提示在 `cmd.exe` 下存在编码兼容风险，已统一改为 ASCII 提示。
- `Invoke-WebRequest` 在当前环境下返回 `System.Byte[]`，已补充统一响应文本转换逻辑，避免健康检查误判。
