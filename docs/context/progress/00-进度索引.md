# 进度索引

## 作用

本目录用于记录阶段性推进和有必要留痕的实施进度，不作为新会话默认全量读取目录。

## 读取规则

- 新会话默认不逐篇读取本目录历史文件
- 连续会话优先结合 `docs/context/current/02-当前任务状态.md` 判断是否需要补读进度记录
- 只有当当前任务需要恢复具体实施脉络时，再按任务编号或日期读取对应进度文件

## 历史文件说明

- 以日期和任务编号命名的文件属于历史进度产物，会持续累积
- 这些文件默认仅用于追溯，不应在主导航中逐篇暴露
- `当前状态`、`下一步建议`、`风险等级`、`下一顺位任务` 等动态信息应只保留在 `docs/context/current/`
