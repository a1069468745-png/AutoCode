# 决策索引

## 作用

本目录用于记录持久性规则、架构或流程决策，不作为新会话默认全量读取目录。

## 读取规则

- 新会话默认不逐篇读取本目录历史文件
- 只有在当前任务涉及规则、架构、流程变更，或需要处理文档冲突时，才读取对应决策文件
- 如果仅需了解当前事实，优先读取 `docs/context/current/`

## 当前建议优先读取

- 与当前阶段或当前治理规则直接相关的最新决策文件
- 被 `docs/governance/` 或 `docs/context/current/` 明确引用的决策文件

## 历史文件说明

- 决策文件默认长期保留，数量会随项目演进缓慢增长
- 主导航应优先指向本索引，而不是逐篇列出历史决策文件
- 阶段切换、任务顺位调整、短期推进状态不应进入本目录，应回写到 `docs/context/current/` 或 `docs/context/progress/`

## 准入与命名规则

- 仅当内容会长期影响规则、架构、流程、兼容性或协作边界时，才进入本目录
- 文件名使用 `YYYY-MM-DD-<主题>-决策.md`
- 若内容主要是在说明“现在做到哪一步”，不得进入本目录
