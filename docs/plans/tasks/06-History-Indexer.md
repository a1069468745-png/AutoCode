# History Indexer 模块任务入口

## 1. 对应总计划

- `HI-01` 到 `HI-04`

## 2. 模块目标

构建 Git 历史事实层，支撑提交、文件、符号、需求号和版本的追溯查询。

## 3. 任务边界

- 范围内：历史扫描、diff 解析、需求号提取、符号级关联、原生 git 回退
- 范围外：不做语义聚类

## 4. 进入门禁

- 已确认 Git 仓来源和历史范围
- 已明确 message 提取规则
- 已确认符号关联依赖的图谱结构

## 5. 关键验证

- 提交历史可回放
- 文件级和符号级关联可追溯
- 回退机制有明确降级行为

## 6. 关键回写

- `docs/context/progress/`
- 提取规则或回退策略变化时更新相关文档
