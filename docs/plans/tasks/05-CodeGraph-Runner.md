# CodeGraph Runner 模块任务入口

## 1. 对应总计划

- `CG-01` 到 `CG-04`

## 2. 模块目标

通过 `CodeGraph` 构建项目代码图谱，落地符号、依赖边、调用边和文件元数据。

## 3. 任务边界

- 范围内：任务服务、受控进程执行、工作目录准备、图谱结果解析、全量与增量索引
- 范围外：不重写图谱解析器，不做复杂影响评分

## 4. 进入门禁

- 已确认当前项目代码仓路径和分支来源
- 已确认图谱输出落地表结构
- 已明确索引失败的记录方式

## 5. 关键验证

- 样例项目可完成图谱索引
- 符号和关系边可落库
- 全量与增量行为可区分

## 6. 关键回写

- `docs/context/progress/`
- 失败或回退策略变动时更新相关说明
