# Dev Agent Service 模块任务入口

## 1. 对应总计划

- `DA-01` 到 `DA-04`

## 2. 模块目标

围绕新需求智能开发辅助，提供结构化分析、影响范围、改造建议和测试建议能力。

## 3. 任务边界

- 范围内：需求解析、相似需求召回、上下文组织、分析报告生成
- 范围外：不直接提交代码变更

## 4. 进入门禁

- 上下文服务与模型网关已具备可用能力
- 已明确当前只做分析，不做代码执行
- 已明确输出边界和可追溯要求

## 5. 关键验证

- 需求可被结构化
- 输出能包含影响范围、风险点、测试建议
- 输出可回溯到上下文来源

## 6. 关键回写

- `docs/context/progress/`
- 智能分析边界变化时更新设计和治理文档
