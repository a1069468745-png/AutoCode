# 数据库与缓存基线模块任务入口

## 1. 对应总计划

- `DB-01` 到 `DB-03`

## 2. 模块目标

建立统一数据库结构、索引策略、缓存键规范和向量检索预留。

## 3. 任务边界

- 范围内：核心表结构、索引、Redis 键规范、Qdrant 集合规划
- 范围外：不实现业务报表、不做复杂向量策略

## 4. 进入门禁

- 基础设施骨架已可用
- 已明确当前变更涉及哪些表或缓存规则
- 已明确验证方式和回写文档

## 5. 关键验证

- SQL 可执行
- 关键索引完整
- 缓存和向量规划可追溯到任务边界

- `docs/context/progress/`
