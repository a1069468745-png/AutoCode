# 基础设施与工程骨架模块任务入口

## 1. 对应总计划

- `INF-01` 到 `INF-05`

## 2. 模块目标

建立统一仓库结构、构建方式、配置规范和部署骨架，为所有后续模块提供复用底座。

## 3. 任务边界

- 范围内：目录结构、Maven 多模块、前端工程骨架、统一配置、Docker Compose 骨架
- 范围外：不实现具体业务模块逻辑

## 4. 进入门禁

- 已确认当前任务属于基础设施模块
- 已读取总执行计划对应片段
- 已明确需要更新的目录和配置文档

## 5. 关键验证

- 构建骨架可用
- 配置覆盖规则明确
- Compose 配置可通过校验

## 6. 关键回写

- `docs/context/progress/`
- 如目录或实施顺序变动，更新总执行计划
