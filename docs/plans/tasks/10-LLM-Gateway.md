# LLM Gateway 模块任务入口

## 1. 对应总计划

- `LLM-01` 到 `LLM-06`

## 2. 模块目标

统一封装第三方模型 API 和本地模型入口，为综合问答和智能分析提供稳定接入层。

## 3. 任务边界

- 范围内：服务骨架、模型配置、统一协议、审计骨架、多模型接入、embedding、问答接入
- 范围外：不直接生成代码补丁

## 4. 进入门禁

- 已明确项目级模型策略来源
- 已明确审计和安全要求
- 已明确是否处于只做接口预留还是正式启用阶段

## 5. 关键验证

- 统一接口稳定
- 配置加密和审计可用
- 模型不可用时具备回退或降级策略

## 6. 关键回写

- `docs/context/progress/`
- 模型策略变化时更新治理或设计文档
