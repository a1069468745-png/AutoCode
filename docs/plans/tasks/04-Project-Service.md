# Project Service 模块任务入口

## 1. 对应总计划

- `PS-01` 到 `PS-04`

## 2. 模块目标

提供项目注册、仓库配置、文档仓绑定、默认分支和模型策略配置能力。

## 3. 任务边界

- 范围内：项目 CRUD、成员与权限映射、索引触发与状态流转
- 范围外：不做批量导入和复杂审批流

## 4. 进入门禁

- 已确认项目模型和状态机的权威来源
- 已明确本次改动影响的实体、接口和文档

## 5. 关键验证

- CRUD 可用
- 项目状态流转可追踪
- 权限查询可支撑网关校验

## 6. 关键回写

- `docs/context/progress/`
- 如状态机变化，更新执行计划或架构说明
