# INF-01 仓库结构初始化 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 创建与总执行计划一致的顶层仓库骨架，并补齐根级说明与忽略规则，为 `INF-02` 和 `INF-03` 提供稳定入口。

**Architecture:** 本任务只处理顶层目录与根级文档，不引入任何业务代码或模块实现。通过最小目录骨架、统一用途说明和基础忽略规则，固定后续模块的落位方式。

**Tech Stack:** `PowerShell`、Markdown、Git ignore 规则

---

## 1. 任务边界

- 范围内：
  - 创建 `backend/`、`frontend/`、`integration-tests/`、`tools/`
  - 创建根级 `README.md`
  - 创建根级 `.gitignore`
- 范围外：
  - 不创建 Maven 模块
  - 不初始化前端工程
  - 不补充业务代码

## 2. 验证方式

- `Test-Path` 验证目录和文件是否存在
- `Get-ChildItem` 验证顶层结构是否与计划一致
- 人工检查 `README.md` 是否覆盖目录用途说明
- 人工检查 `.gitignore` 是否覆盖基础开发输出

## 3. 回写目标

- `docs/context/current/02-当前任务状态.md`
- `docs/context/progress/2026-05-28-INF-01-仓库结构初始化进度.md`
- 如需切换阶段状态，再更新 `docs/context/current/01-当前阶段与目标.md`
