# INF-03 前端工程骨架 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `frontend/web-console` 下初始化 `Vue 3 + TypeScript + Vite + Element Plus + Pinia + Vue Router` 工程骨架，并预留路由、状态管理与请求层入口。

**Architecture:** 使用 `Vite` 作为构建入口，保留轻量级应用外壳，只交付基础布局、路由容器、Pinia 根仓库和 Axios 请求封装，不实现具体业务页面。目录结构优先服务后续 `FE-01` 到 `FE-03` 的扩展，而不是提前堆叠功能。

**Tech Stack:** `Vue 3`、`TypeScript`、`Vite`、`Element Plus`、`Pinia`、`Vue Router`、`Axios`、`pnpm`

---

## 1. 任务边界

- 范围内：
  - 初始化 `frontend/web-console`
  - 安装 `Element Plus`、`Pinia`、`Vue Router`、`Axios`
  - 预留 `router`、`stores`、`api`、`layouts` 目录
  - 保证 `pnpm install`、`pnpm build` 可执行
- 范围外：
  - 不实现业务查询页面
  - 不接入真实后端 API
  - 不实现登录态与权限逻辑

## 2. 验证方式

- `pnpm.cmd install`
- `pnpm.cmd build`
- 人工检查 `src/main.ts`、`src/router/`、`src/stores/`、`src/api/` 是否已预留
